# Cypress Automation Testing – OrangeHRM

## 📌 Overview
This project contains end-to-end (E2E) automated test scenarios using Cypress for the OrangeHRM web application.

## 🧪 Test Scenarios
- Login & Logout  
- Add Employee  
- Delete Employee  
- Job Title Management  
- My Info Updates  
- Regression Testing  

## 🛠 Tools Used
- Cypress  
- JavaScript  

## 📁 Project Structure
- cypress/e2e → test cases  
- fixtures → test data  
- screenshots → test results  
- videos → execution recordings  

## 📊 Results
- Successfully executed multiple test scenarios  
- Validated core system functionality  
- Detected and handled failed test cases
